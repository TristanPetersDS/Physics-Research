Colorblind-safe palette — pick a highlight family

Locked: banner F1, body F, UH Mānoa green (#024731) as primary, UH gold (#C8A464) as accent, off-white #F5F4EE background. Guidelines forbid red+green pairing, so the existing crimson Highlight (#8B1A1A) must be replaced. Each palette below shows the same F1+F mockup so the difference is purely the highlight family. All four are deuteranopia / protanopia / tritanopia safe.

P1

Institutional — UH green + gold + LLNL navy

The most conservative pick. LLNLNavy #003594 is already in the spec; promote it from "logo-only" to the highlight role. Three institutional colors, zero brand-foreign additions. Calm and authoritative.

P4

Multi-accent — UH green + gold + full Okabe–Ito for figures

Reserve gold for the hero, then use the full Okabe–Ito qualitative palette inside figures: vermillion, sky-blue, blue, reddish-purple, yellow. Best when the data plots themselves carry signal across several categories (e.g. the 5/50/150 mm crossover curves). Slightly busier; needs disciplined application.

Hidden question: what energy do you want the headline to carry? P1 navy reads "authoritative / institutional," P2 vermillion reads "discovery / read me," P3 magenta reads "modern / methods-forward," P4 Okabe–Ito reads "data-first / plot-driven." Each plays the colorblind-safety card equally well.

Independent of palette: non-color encodings (markers ●▲■, line styles, labels) will be layered on every multi-category plot per the guidelines — no figure will rely on color alone.
